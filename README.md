# Riftwalkers

**Riftwalkers** is a browser-based game built using [Phaser 3](https://phaser.io/). Players traverse multiple dimensions, face challenges, solve puzzles, and complete quests in order to return to their own reality. Each dimension presents unique obstacles, enemies, and rewards, making every journey a new adventure.
